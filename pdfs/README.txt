To set up your chatbot's knowledge base:

1. Place your PDF file(s) in this directory (pdfs/)
2. Name your PDF file something descriptive (e.g., "manual.pdf", "guide.pdf")
3. Restart the backend server
4. The system will automatically process your PDF and make it available to the chatbot

Supported formats:
- PDF files (.pdf)

The chatbot will be able to answer questions based on the content of your PDF document(s).
